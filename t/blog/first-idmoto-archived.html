Update on IDMOTO

14.05.2025

Hey!

Today marks the day when the first version of the IDMOTO app is officially archived as a public release. I’ve decided that this is the perfect point to raise the bar and turn it into something more serious and improved. That’s why the original version is now available on GitHub under the name idmoto-local-storage. The name refers to the browser’s local storage, which was the main storage method for all data entered into the app to create a car profile. Now, the app is being rebuilt using Next.js and Google Firebase, so that every user will be able to log in and create actual profiles that can be shared online — or even in the real world through a QR code sticker on their car.
The story is just beginning :)
